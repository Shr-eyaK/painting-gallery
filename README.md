# Painting Gallery

A simple web application to display and manage a gallery of paintings. Built using PHP, SQL, HTML, and CSS.

## Features
- View all paintings in the gallery
- Add new paintings
- Delete existing paintings
- Clean and responsive layout

## Technologies Used
- PHP
- MySQL / SQL
- HTML & CSS
